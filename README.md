(this bot is for personal use so i will not fix or explain any issues!)

Setup:
create a file named bot_config.py with
```python
# Base Settings (needed  to run)
TOKEN = ""
DEBUG = True 
COMMAND_PREFIX = "!" 

# Extra Settings 
OWNER_USERIDS = {}

# Google forms
BASE_FORM_ID = ""
SERVICE_ACCOUNT = {}
```
